# Status and Roadmap

## Current status

**Early-stage methods development.**

This repository supports a biomedical imaging methods scaffold focused on simulated PET image noise, lesion segmentation, radiomics feature stability, and downstream model robustness. It remains exploratory until a public PET/CT dataset, perturbation protocol, and reproducible feature-extraction workflow are selected.

## Public-facing status language

Use the following wording in CV, portfolio, and public research-status outputs:

> PET noise and radiomics robustness scaffold focused on simulated PET noise, lesion segmentation, and radiomics feature stability; public dataset, perturbation protocol, and validated results remain pending.

## Near-term roadmap

1. Select a public PET/CT dataset and document access terms.
2. Define simulated-noise perturbation levels.
3. Specify segmentation and feature-extraction assumptions.
4. Create radiomics feature-stability table templates.
5. Add a reproducible baseline workflow.
6. Draft a limitations note separating exploratory robustness work from validated clinical claims.

## Validation gates

- Do not claim clinical robustness until public data and reproducible results exist.
- Keep simulated-noise assumptions explicit.
- Preserve data provenance, preprocessing choices, and feature-extraction settings.

## Roadmap priority

Medium priority as a biomedical AI evaluation scaffold after ECG benchmark design is stabilized.
