# Project Description

This repository contains an early-stage biomedical imaging robustness workspace for simulated PET image noise, lesion segmentation, radiomics feature stability, and downstream model-reliability evaluation. It is intended as a reproducible methods scaffold, not as a validated clinical robustness claim.

The next project gates are public PET/CT dataset selection, perturbation protocol definition, segmentation assumptions, feature-extraction settings, feature-stability tables, and a limitations note.

## Short description

PET noise and radiomics robustness scaffold for reproducible biomedical imaging feature-stability evaluation.

## Scope boundary

Do not claim clinical robustness, diagnostic performance, or model reliability until public data, reproducible methods, and validated result tables exist.
