# Project Manifest

## Repository role

This repository is a biomedical imaging robustness workspace for simulated PET noise, lesion segmentation, radiomics feature stability, and downstream model-reliability evaluation.

## Current status

Early-stage methods development.

## Public CV wording

PET noise and radiomics robustness scaffold focused on simulated PET noise, lesion segmentation, and radiomics feature stability; public dataset, perturbation protocol, and validated results remain pending.

## Primary objects

- Public PET/CT dataset manifest
- Simulated-noise perturbation protocol
- Segmentation assumptions
- Radiomics feature-extraction settings
- Feature-stability result tables
- Reproducible baseline workflow
- Limitations and exploratory-claim note

## Roadmap priority

Priority 6. Biomedical AI robustness scaffold after ECG benchmark design is stabilized.

## Validation boundary

Do not claim clinical robustness, diagnostic performance, or model reliability until public data, reproducible methods, and result tables exist.
