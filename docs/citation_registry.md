# Citation Registry

## Verification policy

All citations must be validated against:

- DOI registry;
- publisher page;
- PubMed where applicable;
- TCIA records;
- official challenge pages.

Unverified claims must be labeled:

UNVERIFIED — POSSIBLE SIMULATION.
