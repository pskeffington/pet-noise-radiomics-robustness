# Repository Structure

```text
pet-noise-radiomics-robustness/
├── analysis/
├── data/
│   ├── manifests/
│   └── derived/
├── docs/
├── figures/
├── manuscript/
├── notebooks/
├── tables/
├── tests/
└── workflows/
```

## Directory intent

### analysis/

Core reproducible scripts.

### data/manifests/

Dataset manifests and checksums.

### data/derived/

Derived non-raw outputs.

### docs/

Project documentation and planning.

### figures/

Publication-ready figure exports.

### manuscript/

LaTeX manuscript workspace.

### notebooks/

Exploratory notebooks only.

### tables/

Reproducible manuscript tables.

### tests/

Validation and reproducibility tests.

### workflows/

Automation and pipeline definitions.
